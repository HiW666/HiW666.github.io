arduino的GPIO初始化

引脚初始化

参数 数据类型 作用 选项
引脚号 int 指定引脚初始化 自己查芯片手册
选择引脚模式 int 指定引脚模式 OUTPUT（0x03）：输出模式
INPUT（0x01）：输入模式
PULLUP（0x04）:上拉
INPUT_PULLUP（0x05）：上拉输入
PULLDOWN（0x08）：下拉
INPUT_PULLDOWN（0x09）：下拉输入
OPEN_DRAIN（0x10）：开漏
OUTPUT_OPEN_DRAIN（0x13）：开漏输出
ANALOG（0xC0）：模拟输入
返回值 无 无 无

arduino的GPIO输出

引脚模式 int 选择输出高低电平 HIGH（0x1）：输出高电平
